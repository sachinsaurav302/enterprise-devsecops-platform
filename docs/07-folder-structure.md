# Folder Structure

## Repository Organization
- application/: application code and services
- architecture/: system and infrastructure architecture documentation
- docs/: project documentation and decision records
- github-actions/: reusable CI/CD workflow definitions
- helm/: Kubernetes packaging and deployment charts
- kubernetes/: Kubernetes manifests and resource definitions
- monitoring/: dashboards, alerts, and observability configuration
- scripts/: utility scripts and automation helpers
- security/: security policies, configurations, and samples
- terraform/: infrastructure as code modules
- database/: database schemas and migration scripts

## Notes
This structure separates source code, infrastructure, automation, and documentation to improve maintainability and governance.
