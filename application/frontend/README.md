# Enterprise DevSecOps Platform - Frontend

This is the frontend user interface for the Enterprise DevSecOps Platform.

## Folder Structure

- `public/`: Static assets (favicon, direct files, etc.).
- `src/`: Frontend source code.
  - `components/`: Reusable/shared UI components.
  - `pages/`: Page-level views and layouts.
  - `services/`: API client services and network request handlers.
  - `hooks/`: Custom React/framework hooks.
  - `context/`: React context providers for global state management.
  - `assets/`: Images, icons, and local media assets.
  - `styles/`: Stylesheets, CSS modules, or CSS-in-JS files.
  - `utils/`: Helper/utility functions.

## Getting Started

### Prerequisites

- Node.js (v18+)
- npm

### Installation

```bash
cd application/frontend
npm install
```

### Running the Application

```bash
# Development mode
npm run dev

# Build for production
npm run build

# Preview production build locally
npm run preview
```
